# Password manager manual

### How to use:

#### Register:

> <p style="text-align: start;">Click register text:</p>
> <img src="./Documentation Media/register_1.png" alt="Register icon 1" style="width: 40em" />
> 
>  &nbsp; 
>
> `A register form will open.`
> 
> <p style="text-align: start;">Fill the form with:</p>
>
> 1. Username
> 2. Password
> 3. Re enter password
> 4. Then click register button to create new usesr profile (The application support mutliple users in the same system)
>
> `* You can click the login text if you have already a acount.`
>
> <img src="./Documentation Media/register_2.png" alt="Register icon 2" style="width: 40em" />
>
> `After clicking the register button an account will be created a message will pop up informing you of your action was successful, the window will close and a login window will open.`

---

#### Login:

> <p style="text-align: start;">To login enter your:</p>
>
> 1. Username
> 2. Password
> 3. Click the remember me checkbox if you want to have the username filled the next time you open the application (optional)
> 4. Login
> 
> <img src="./Documentation Media/login.png" alt="Login icon" style="width: 40em" />
>
> * `If your credentials are correct: you will get a notification and login window will close, the main window of the application will open.`
> * `If your credentials are incorrect: you will be informed with a notification (you have three tries before locking the Login function for ten minutes)`
>
> &nbsp; 
>
> `Main window:`  
> <img src="./Documentation Media/main_window.png" alt="Main application window icon" style="width: 40em" />

#### Edit item:
> <p style="text-align: start;">Item fields explanation:</p>
>
> 1. Item name
> 2. Item website
> 3. Username, email or phone number
> 4. Copy website (button)
> 5. Copy username, email or phone number (button)
> 6. Copy password (button, the value on clipboard lasts for five seconds)
> 7. Edit item
> <img src="./Documentation Media/display-item-func.png" alt="Item field explanation image" style="width: 60em" />
>
>  &nbsp;  
>
> <p style="text-align: start;">While you are in edit mode you can change any value, click the save button to save the changes or click cancel to discard</p>
> <img src="./Documentation Media/editing.png" alt="Edit mode example image" style="width: 40em" />

#### Delete item:
> <p style="text-align: start;">To delete an item, click the edit button of the item to enter the edit mode (num. 7)</p>
> <img src="./Documentation Media/display-item-func.png" alt="Delete explanation image" style="width: 60em" />
>
> &nbsp;   
>
> <p style="text-align: start;">Click the delete button to delete the item or click the cancel button to discard</p>
> <img src="./Documentation Media/editing.png" alt="Delete action explanation image" style="width: 40em" />
>
> &nbsp;   
>
> <p style="text-align: start;">After clicking the delete button, a pop up window will saw up asking for confirmation. Press Yes button to delete, No or X to cancel</p>
